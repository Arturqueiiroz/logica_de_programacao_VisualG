# Lógica de Programação com VisualG

Repositório criado para reunir exercícios, algoritmos e práticas desenvolvidas em **Lógica de Programação** utilizando **VisualG** e **Português Estruturado**.

## Sobre o projeto

Este repositório tem como objetivo armazenar exercícios resolvidos durante os estudos de lógica de programação, com foco no desenvolvimento do raciocínio lógico, na construção de algoritmos e na prática com estruturas fundamentais da programação.

Os exercícios foram organizados em listas e pastas, facilitando a visualização e a consulta de cada conteúdo estudado.

## Conteúdos praticados

- Entrada e saída de dados
- Declaração de variáveis
- Operações matemáticas
- Estruturas condicionais
- Estruturas de decisão
- Exercícios de fixação em VisualG
- Algoritmos em Português Estruturado

## Estrutura do repositório

Atualmente, o repositório está organizado em pastas como:

- `faccat`
- `listaDeExercico01`
- `listadeexerciciosEstruturadeDecisao`
- `manzano`

Cada pasta reúne exercícios e atividades de prática relacionados a diferentes listas ou materiais de apoio.

## Tecnologias e ferramentas

- **VisualG**
- **Português Estruturado**

## Objetivo

Este repositório faz parte da minha jornada de aprendizado em programação, servindo como apoio para revisar conteúdos, praticar exercícios e acompanhar minha evolução na lógica de programação.
